
# Face Gaze Tracker

Using flask for run at localhost and droidCam module for the camera, you need setup the venv and install requirements

### Setup the VENV

```bash
  python -m venv venv
```
### Install the Requirements

```bash
  pip install -r requirements.txt
```

### Run app

```bash
  python app.py
```

### Setup Integrated Camera fro Web (WebRTC)

You must allow the camera permission for display the processed image of cheating detection
